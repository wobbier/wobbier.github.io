How This Site Works

This site is a static build powered by Metalsmith and Node.js. Every page starts as a markdown file with some front matter and gets rendered through a Handlebars layout. Adding a new post is just dropping a file into src/blog: 
---
layout: blog-post.html
title: "My New Post"
date: July 13th, 2026
slug: "my-new-post"
---
Post content goes here.
